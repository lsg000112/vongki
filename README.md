# 봉키(VONGKI)

## 배경

코로나19와 봉사활동의 대입 미반영 등의 여파로 MZ세대를 비롯한 봉사활동 참여율이 급감했습니다.
봉키는 MZ세대의 자원봉사 참여에 약간의 재미를 가미하여 자발적인 봉사활동 참여가 이루어지도록 만드는 플랫폼입니다.
